    Parameters

    components

    URLComponents describing path relative to future origin of the transport.
    headers

    Dictionary of the HTTP headers attached to the Request. Default value is nil.
    method

    Method option listed above. Default value is .get.
    body

    Optional piece of Data attached as a body of the HTTP request. Default value is nil.
    mode

    Lifecycle which will be used to manage (or not to) background execution.
