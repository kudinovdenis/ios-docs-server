Hestia

Swift framework contains endpoint for communication with IDP @ Acronis.

Dependencies

  • Neuro — https://git.acronis.com/projects/AMB/repos/neuro
  • Depot — https://git.acronis.com/projects/AMB/repos/depot
  • Alchemy — https://git.acronis.com/projects/AMB/repos/alchemy
  • Hermes — https://git.acronis.com/projects/AMB/repos/hermes

How to Build

  • Bootstrap all the dependencies using Carthage.
  • Open .xcproject in XCode and build any desired target.

How to Run

Nothing to Run.

Usage

import Neuro
import Depot

import Alchemy
import Hermes

import Foundation

let transport = FoundationHTTP(backgroundSessionIdentifier: self.invocation.debugDescription)
let address = FoundationHTTP.Address(string: "https://trueimage.acronis.com/")!

let explorer = HermesExplorer()
let receipt = HermesExplorer.Receipt(input: address, transport: transport)

let hermes = try! explorer.explore(with: receipt)

let login = "artem.olkov@acronis.com"
let origin = try! hermes.origin(for: login)

let hestiaExplorer = HestiaCloudExplorer()
let hestiaCloudReceipt = HestiaCloudExplorer.Receipt(root: address, input: origin, transport: transport)

let hestia = try! hestiaExplorer.explore(with: hestiaCloudReceipt)

let hestiaLogin = login
let hestiaPassword = "qwe123QWE"

let credentials = try! hestia.authorize(login: hestiaLogin, password: hestiaPassword)
let _ = try! hestia.authorize(with: credentials)
