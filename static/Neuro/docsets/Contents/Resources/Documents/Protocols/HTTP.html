HTTP

public protocol HTTP: Transport

A protocol defining OSI’s HTTP/S application abstraction layer.

  • task(for:on:)

    Fabric method provides assembled and verifyed entity ready to be sent over the network.

    Important

    This does not actually start any network activity. Instead, you’ve got only assembled token for further perform in any suitable place.

    Throws

    Any custom error declared by code that implements this protocol. In common this error reflects input validation error, but it could mean any error during any custom behaviour implemented in concrete HTTP transport.

    Declaration

    Swift

    func task(for request: Request, on address: Address) throws -> Task

    Parameters

    request

    Common input entity which you should have describing your HTTP request including headers, body, method and anything specific for concrete implementation.
    address

    Resource address in terms of this concrete HTTP transport.

    Return Value

    Mentioned Task entity for further usage.

  • perform(task:)

    Actually performs Task provided in the previous step.

    Throws

    Any custom error declared by code that implements this protocol. Reflects any error during the network request related to HTTP or lower OSI level.

    Declaration

    Swift

    func perform(task: Task) throws -> Response

    Parameters

    task

    Task entity you’ve got as result of according fabric method.

    Return Value

    Actual common Response of concrete HTTP transport.

  • status(of:)

    Current information for specified Task.

    Important

    Reflects status as it is for the moment when method was called. Could and mostly should be called from any thread that is not the same where was called perform() method.

    Throws

    Any custom error declared by code that implements this protocol.

    Declaration

    Swift

    func status(of task: Task) throws -> Status

    Parameters

    task

    Task entity you’ve got as result of according fabric method.

    Return Value

    Status entity describing neccesary information of the Task specific for concrete HTTP transport implementation.

  • cancel(task:)

    Does cancellation and invalidation of the provided Task.

    Important

    Should be called only once. In other case error will be thrown.

    Throws

    Any custom error declared by code that implements this protocol. Does not throw when success.

    Swift
